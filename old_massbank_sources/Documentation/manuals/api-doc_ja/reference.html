MassBank WEB-API リファレンス   English


  • スキーマ(WSDL)
    • http://www.massbank.jp/api/services/MassBankAPI?wsdl

    取得系メソッド
    getInstrumentTypes メソッド

    MassBankに登録されているデータの分析機器種別すべてを取得します。
    getRecordInfo メソッド

    MassBankレコード情報（ピークデータ、分析条件等すべて含む）を取得します。
    getPeak メソッド

    ピークデータを取得します。


    検索系メソッド
    searchSpectrum メソッド

    類似スペクトルを検索します。
    searchPeak メソッド

    指定したm/zのピークが出現するスペクトルを検索します。
    searchPeakDiff メソッド

    指定したm/z差のピークが出現するスペクトルを検索します。
    searchSubstructure メソッド    ・・・ 後日公開予定

    化学構造による化合物検索を行います。


    バッチ処理関連メソッド  NEW
    execBatchJob メソッド

    バッチ処理に一括検索を実行します。
    getJobStatus メソッド

    指定したジョブIDのステータスを取得します。
    getJobResult メソッド

    指定したジョブIDの結果を取得します。
