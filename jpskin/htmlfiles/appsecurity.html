The security checks of Java blocked MassBank applets

To all the MassBank users who installed Java 7 Update 51, we apologize for their inconvenience that mass spectra are not displayed because the security checks of Java blocked MassBank applets.

To solve the problem without decreasing the security level of Java, we are now going to embed the Java code signing into the MassBank source codes.

Escape the problem by using “Exception Site List” that is introduced

Before a new version of the MassBank embedded with the code signing could be released, we ask the MassBank users to escape the problem by using “Exception Site List” that is introduced in the release of the update 51.  Please refer to the following URL
http://www.java.com/en/download/faq/exception_sitelist.xml?printFriendly=true
